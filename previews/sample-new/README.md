# Aliens Last Night Website

Static website for Aliens Last Night.

## Files

- `index.html` — homepage
- `about.html` — about page
- `find.html` — channels page
- `partnerships.html` — partnerships page
- `og.html` — OpenGraph preview page
- `styles.css` — site styles

## Local preview

```bash
python3 -m http.server 8765
```

Then open http://127.0.0.1:8765/.

## Deployment

This is a static site deployed from the repository root with GitHub Pages.
